# Description Engine

## Purpose

Use this file for 묘사 방식, 디테일, 감각, 전투, 공간 동선, 아이템 판독, social gaze, and emotional description.

## Core Rule

Description is not decoration. It must help the reader understand action, danger, reward, emotion, or spatial position.

Pick one primary description axis per scene and one secondary axis only when needed.

## Description Axes

### Body Sense

Use for fear, injury, exhaustion, growth, poison, mana, hunger, shame, and instinct.

Pattern: body reaction -> immediate action or interpretation.

### Space Route

Use for dungeon movement, escape routes, chase scenes, school corridors, battlefield layouts, or office/social positioning.

Pattern: landmark -> movement -> constraint -> next choice.

Do not list directions as isolated fragments. Turn route data into sentences.

### Threat Detection

Use sound, smell, gaze, temperature, silence, vibration, or wrongness. Pick one.

Pattern: detection -> doubt -> confirmation -> decision.

### Item And Ability Readout

Use for status windows, items, skills, tools, weapons, contracts, documents, or money.

Pattern: readout -> usefulness -> cost/risk -> desire.

### Social Gaze

Use for school, academy, guild, company, courtroom, livestream, martial sect, noble court, or public ranking scenes.

Pattern: public gaze -> protagonist choice -> visible proof -> changed reaction.

### Combat Impact

Use action beats in order:

1. Setup.
2. Movement.
3. Impact.
4. Consequence.
5. Reaction or new threat.

Keep metaphors sparse during impact. Let body, sound, and result carry the scene.

### Emotion Concealment And Release

For suppressed emotion, use small behavior: avoiding eye contact, checking a phone, gripping a strap, swallowing words.

For release, show action first and name the feeling only if needed.

## Detail Budget

- Minor object: one phrase.
- Important clue: one sentence plus one consequence.
- New place: two to four sentences tied to movement.
- Monster/enemy entrance: one sensory detail, one silhouette/action detail, one threat detail.
- Reward/item: status/readout plus desire or risk.

## Bad Signs

- The description could be removed without changing action or emotion.
- The scene uses three or more sensory channels without a clear reason.
- The same emotion is described repeatedly with new metaphors.
- The reader knows the mood but not where the character is standing.
