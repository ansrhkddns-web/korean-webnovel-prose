# Genre Persona

## Purpose

Use this file when drafting or planning across genres while keeping commercial webnovel quality consistent.

Do not imitate any specific author. Use genre promises, reward rhythms, and scene engines.

## Dungeon / Hunter / System

- Promise: danger zone entry, detection, fight, readout, reward, deeper signal.
- Opening pressure: door, gate, dungeon smell, alert, monster trace, missing person, item reaction.
- Mid reward: status change, item, map, clue, survival proof.
- Ending hook: reward points to a deeper zone or reality breach.
- Description priority: space route, threat detection, item readout, body sense.

## Apocalypse / Survival / Base

- Promise: scarcity, safety breach, resource choice, group trust, contamination.
- Opening pressure: supplies run out, safe place fails, infected trace, betrayal, weather, locked route.
- Mid reward: food, tool, safe room, ally trust, new rule.
- Ending hook: safety turns out conditional or contaminated.
- Description priority: body sense, resource detail, threat detection, group reaction.

## Game / Tower / Player

- Promise: rule discovery, exploit, ranking, stage progression, audience reaction.
- Opening pressure: quest condition, penalty timer, failed guide, rival ranking.
- Mid reward: exploit, experience, skill, title, public/audience response.
- Ending hook: next floor, hidden condition, new challenger.
- Description priority: system readout, rule testing, public reaction.

## Martial Arts / Cider / Hierarchy

- Promise: insulted low-status character proves hidden strength.
- Opening pressure: public disrespect, sect hierarchy, duel demand, master/disciple debt.
- Mid reward: one clean reversal, recognition, opponent panic.
- Ending hook: higher-ranked figure notices or intervenes.
- Description priority: social gaze, combat impact, cold restraint.

## Modern Professional / Success

- Promise: competence turns into measurable social or financial gain.
- Opening pressure: work crisis, deal, lawsuit, surgery, broadcast, investment timing.
- Mid reward: solved case, contract, number, promotion, public proof.
- Ending hook: a bigger stakeholder enters the board.
- Description priority: procedural detail, money/contract/item readout, public reaction.

## School / Academy / Group Conflict

- Promise: rumor pressure and social hierarchy are overturned by evidence, skill, witness, or performance.
- Opening pressure: classroom gaze, group chat, teacher pressure, bully move, test/ranking.
- Mid reward: evidence, witness, public change, competence reveal.
- Ending hook: new frame, hidden recording, adult intervention, dungeon/ability spillover.
- Description priority: social gaze, teen casual dialogue, body shame/fear, public reaction.

## Fantasy / Territory / War

- Promise: resource, terrain, politics, command, war pressure, long-term payoff.
- Opening pressure: invasion, tax, noble threat, famine, troop gap, succession conflict.
- Mid reward: alliance, resource, military trick, territory improvement.
- Ending hook: war choice, political trap, enemy movement.
- Description priority: map/space, logistics, public command, cost.

## Mixed Genre Protocol

Pick one main promise and one support flavor.

Example: dungeon + school means school pressure should push the protagonist into dungeon action or make dungeon rewards useful in school. Do not let support flavor replace the main promise for more than one chapter.
